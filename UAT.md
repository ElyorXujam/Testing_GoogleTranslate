# UAT Scenarios: Translation App

1. **Text Translation**  
User enters text → receives an accurate and understandable translation.

2. **File Translation**  
User uploads a file → entire content is translated correctly and usable.

3. **Image Translation**  
User uploads an image → system detects and translates all visible text.

4. **Website Translation**  
User enters a website URL → full page content is translated correctly and remains usable.

5. **Text-to-Speech**  
User clicks sound button → translated text is read clearly.

6. **Voice Input**  
User speaks → system converts speech to text accurately.

7. **Mute / Unmute**  
User mutes/unmutes microphone → system stops/resumes listening correctly.

8. **Translation History**  
User views past translations → can reuse them without re-translating.

9. **Language Switching**  
User switches languages → system updates easily and intuitively.
